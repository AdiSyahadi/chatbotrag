System Prompt

Atur instruksi dan kepribadian chatbot. Perubahan langsung berlaku di percakapan berikutnya.

Edit System Prompt
Cara kerja: System prompt adalah instruksi yang diberikan ke AI sebelum menjawab pertanyaan pengguna. Di sinilah kamu menentukan nama, gaya bicara, dan aturan chatbot.
Wajib ada: Placeholder {context} (isi dokumen) dan {question} (pertanyaan user) harus tetap ada di dalam prompt.

Tips Menulis System Prompt

Tips Membuat Prompt

  • Jelaskan persona chatbot (siapa dia, apa tugasnya).
    Contoh: "Kamu adalah Chatbot Desa, asisten virtual resmi untuk layanan informasi pemerintahan Desa."
  • Tetapkan batasan agar bot tidak menjawab di luar konteks dokumen.
    Contoh: "Jawab HANYA berdasarkan informasi (Konteks) yang diberikan."
  • Atur gaya bahasa (formal, santai, ramah).
    Contoh: "Gunakan bahasa Indonesia yang baik, ramah, dan sopan."

4. Jangan hapus placeholder
{context} akan diisi otomatis dengan isi dokumen yang relevan.
{question} akan diisi otomatis dengan pertanyaan pengguna.
